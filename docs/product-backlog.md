Product BackLog

Histórias de usuário - Prioridade Alta:

US01 — Acessibilidade por Voz

Como pessoa com deficiência visual,
quero navegar no app usando comandos de voz,
para conseguir utilizar todas as funções sem depender da tela.

Critérios de Aceitação:

-O usuário consegue ativar o comando por voz pelo microfone do celular.

-O app executa ações básicas: navegar, selecionar treinos, iniciar/pausar.

-Resposta sonora de confirmação após cada ação.

Prioridade: Alta

US02 — Leitura de Tela (Screen Reader)

Como usuário com deficiência visual,
quero que todos os elementos tenham suporte a leitores de tela,
para que eu consiga entender botões, textos e descrições.

Critérios de Aceitação:

-Todos os botões possuem labels acessíveis.

-Imagens têm descrição alternativa.

-Testado com TalkBack/VoiceOver.

Prioridade: Alta

US03 — Treinos Adaptados

Como pessoa com mobilidade reduzida,
quero selecionar treinos adaptados à minha condição,
para evitar exercícios que não posso realizar.

Critérios de Aceitação:

-Treinos possuem etiqueta: “Adaptado para cadeira de rodas”, “Baixa mobilidade”, etc.

-O usuário consegue filtrar treinos por tipo de limitação.

-Cada treino exibe instruções claras e seguras.

Prioridade: Alta

Histórias de usuário - Prioridade Média:

US06 — Contraste Alto e Ajuste de Fonte

Como pessoa com baixa visão,
quero ativar modo de alto contraste e fontes maiores,
para melhorar minha leitura e navegação.

Critérios de Aceitação:

-App tem modo alto contraste.

-Ajuste de tamanho da fonte.

-Preferências salvas no dispositivo.

Prioridade: Média

US07 — Legendas e Libras

Como usuário com deficiência auditiva,
quero vídeos com legenda e opção de intérprete em Libras,
para entender completamente as explicações dos exercícios.

Critérios de Aceitação:

-Legendas sincronizadas.

-Botão “Ativar Libras”.

-Vídeo com janela de intérprete.

Prioridade: Média

US08 — Feedback de Execução

Como usuário iniciante ou PCD,
quero receber feedback sobre a execução do exercício,
para evitar erros e lesões.

Critérios de Aceitação:

-Feedback por texto, ícones ou áudio.

-Avisos: “muito rápido”, “corrija postura”, etc.

-Deve ser desativável.

Prioridade: Média

Histórias de usuário - Prioridade Baixa:

US09 — Gamificação Inclusiva

Como usuário,
quero ganhar badges inclusivos ao completar treinos,
para me sentir motivado.

Critérios de Aceitação:

-Sistema de pontos.

-Badges por metas alcançadas.

-Representações inclusivas.

Prioridade: Baixa

US10 — Mapa de Acessibilidade das Academias

Como usuário PCD,
quero ver quais academias da minha cidade são acessíveis,
para escolher onde treinar fisicamente.

Critérios de Aceitação:

-Mapa mostrando academias acessíveis.

-Filtros: rampa, banheiro adaptado, equipamentos inclusivos.

-Avaliações de acessibilidade por usuários.

Prioridade: Baixa
